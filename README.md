# Going Cloud Native

`PRs are much appreciated!`

## Quickstart

Follow the installation process [here](https://gohugo.io/getting-started/installing).

Clone the repo

```Shell
git clone https://github.com/omar16100/Going-Cloud-Native.git
```

In the directory

To compile

```Shell
hugo
```

To start the server

```Shell
hugo server
```

For other options

```Shell
hugo --help
```
